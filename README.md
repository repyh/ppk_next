## Getting Started

1. Clone repository
```bash
git clone https://github.com/your-username/katalog-umkm-kelapa-dua.git
cd katalog-umkm-kelapa-dua
```

2. Install dependencies
```bash
npm install
```

3. Run development server
```bash
npm run dev
```

4. Buka browser dan akses `http://localhost:3000`
